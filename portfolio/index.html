PROJETS

Section consacrée à la présentation de certains projets principaux. Ces projets sont pour la plupart développés sur les technologies .NET avec notamment C#, WPF (Windows Presentation Foundation) et UWP (Universal Windows Platform).
N'hésitez pas à consulter les vidéos disponibles.

PIEUVRE ELECTRIQUE

Création d'un outil permettant le dessin d'une implantation électrique de maison, puis la réalisation du dessin des pieuvres électriques, la génération des schémas de câblage des boites de pieuvre et le dessin d'un plan de tableau électrique

GENERATION DEVIS

Il s'agit d'une application UWP, permettant la génération automatique de devis, en liaison avec les implantations électriques réalisées avec le logiciel de pieuvres électriques

POC REFONTE JCJV

Il s'agissait de proposer une idée d'architecture, pour la refonte du projet Je chiffre et Je vend de Legrand. Avec une partie application Desktop, une synchronisation sur serveur distant et une partie application mobile. Il s'agit donc d'un simple POC développé rapidement pour la démonstration de cet architecture

DESIGN D'INTERFACES

Cette section est un portfolio, non exhautif, d'interfaces réalisées soient pour des projets concrets, soit des POC

REC

Il s'agit d'une recherche de design d'interface pour une application mobile embarquée sur Raspberry PI
Cette recherche d'interface a débouché sur un POC développé en UWP (Universal Windows Platform) tournant sur un Windows IOT sur le Raspberry PI 3. Le tout connecté à un capteur de mouvement pour détecter l'angle d'inclinaison du volant de la voiture

GESTION DE PRODUCTION

Il s'agit d'une solution de gestion de production (Stocks, commandes fournisseurs, expéditions, réceptions) et planning spécifique au besoin de Stevia. En liaison avec le logiciel de dessin de pieuvre.

Enibox

Il s'agit d'un projet web développé en python, initialement prévu pour fonctionner sur Raspberry PI mais finalement porté sur un serveur web centralisé. Il permet d'estimer la consommation électrique et en eau d'un logement, à partir de l'étude thermique demandé par la norme RT2012.
